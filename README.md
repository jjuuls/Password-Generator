# Password Generator

A Python command-line tool that generates customizable secure passwords.

## Features
- Choose password length
- Include uppercase letters
- Include lowercase letters
- Include digits
- Include special characters
- Guarantees selected character types are included
- Input validation
- Option to generate multiple passwords

## Technologies Used
- Python
- random
- string

## How To Run

```bash
python password_generator.py
